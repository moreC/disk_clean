# C盘大文件监控工具

一个功能强大的Windows磁盘清理和监控工具，具有GUI界面和智能缓存功能。

## 功能特性

### 🗂️ 大文件扫描
- 扫描指定大小以上的文件（默认10MB）
- 支持完整扫描和快速扫描（使用缓存）
- 按大小或修改时间排序
- 文件类型过滤

### 🔍 可疑无用文件检测
- 自动识别临时文件、缓存、日志、备份文件
- 按文件类型过滤
- 显示文件年龄（天数）
- 安全删除功能

### 📊 目录统计
- 树形结构展示目录大小
- 支持包含/排除系统目录
- 实时显示C盘使用情况对比
- 颜色标记大目录（>1GB红色，>100MB橙色）

### 💾 智能缓存
- 文件缓存：避免重复扫描未变动的文件
- 目录缓存：加速目录大小统计
- 缓存导入/导出
- 自动清理过期缓存

## 文件说明

| 文件 | 说明 |
|------|------|
| `disk_monitor_gui.py` | 主程序 - GUI监控工具 |
| `disk_checker.py` | 大文件检查器（命令行版） |
| `disk_cleaner.py` | 磁盘清理工具 |
| `task_manager.py` | 定时任务管理 |
| `cleanup_plan.py` | 清理计划生成器 |
| `README定时任务.md` | 定时任务使用说明 |

## 使用方法

### 运行GUI工具
```bash
python disk_monitor_gui.py
```

### 命令行工具
```bash
# 大文件检查
python disk_checker.py

# 磁盘清理
python disk_cleaner.py --min-size 50

# 任务管理
python task_manager.py --query
```

## 系统要求

- Windows 7/10/11
- Python 3.6+
- 管理员权限（可选，用于访问系统目录）

## 注意事项

1. **首次扫描**：建议使用"完整扫描"建立缓存
2. **日常使用**：使用"快速扫描"大幅提升速度
3. **系统目录**：扫描C盘时建议勾选"包含系统目录"获取完整数据
4. **删除文件**：删除操作不可恢复，请谨慎操作

## 缓存文件

- `data/scan_cache.json` - 文件扫描缓存
- `data/dir_cache.json` - 目录大小缓存
- `data/last_scan.json` - 上次扫描结果
